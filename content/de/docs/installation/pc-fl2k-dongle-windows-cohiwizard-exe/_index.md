## Installationsanleitung

Diese Anleitung führt dich durch die Installation für folgende Kombination:
- **Control-Computer:** PC
- **Signalwandler:** fl2k-Dongle
- **Betriebssystem:** Windows
- **Software:** COHIWizard exe

### Schritt 1: Voraussetzungen

- Windows 10 oder neuer
- fl2k-Dongle USB Device
- Administratorrechte erforderlich

### Schritt 2: fl2k-Dongle Treiber installieren

Lade die Treiber für deinen fl2k-Dongle herunter...

### Schritt 3: COHIWizard herunterladen

Lade die neueste exe-Version von [hier](https://github.com/radiomuseum/cohiradia) herunter.

### Schritt 4: Installation durchführen

Starte die exe und folge dem Installationswizard...

### Nächste Schritte

Nach der erfolgreichen Installation kannst du mit der Konfiguration beginnen.
